# 🎨 AI Painter — 像素绘画专精技能

让 Claude Code 成为像素画家：搜索像素风参考图 → 设计分层像素画 → 在真实画布上逐格绘制。

**不是用代码"生成"图片，而是像人一样移动鼠标、点击格子、一格一格画出来。**

## 演示

```
用户："帮我画一只蚊子"
  → WebSearch 搜索像素风参考图
  → 分层设计（轮廓 → 结构 → 细节色彩）
  → CDP 浏览器打开画板
  → 逐颜色分组绘制 ~240 像素
  → 🦟 32×32 蚊子完成！
```

![mosquito preview](demo/mosquito-32.png)

## 功能

| 模块 | 说明 |
|------|------|
| 🔍 像素优先检索 | 先搜索 `pixel art sprite` 参考，了解同行如何简化 |
| 🧱 三层设计法 | 轮廓(Silhouette) → 结构(Structure) → 细节色彩(Detail) |
| 🖼️ 多尺寸画板 | 16×16 / 24×24 / 32×32 / 48×48，AI 按对象复杂度自选 |
| 🎨 36 色调色板 | 无色系 8 + 暖色 8 + 冷色 8 + 特殊色 12，每色系浅/中/深三阶 |
| 🖱️ 真实鼠标绘制 | CDP 协议驱动的浏览器画板，鼠标逐格点击 |
| 📸 PNG 导出 | Canvas 渲染高清像素图并下载 |

## 安装

```bash
# 复制到 Claude Code 技能目录
cp -r ai-painter ~/.claude/skills/
```

## 画板特性

`assets/pixel-canvas.html` — 一个功能完整的像素画编辑器：

- 16/24/32/48 四个尺寸可选
- 36 色调色板（按色系分组）
- 点击 + 拖拽绘制
- Ctrl+Z 撤销
- 导出 PNG
- 每个格子有 `data-row` / `data-col` 属性，CDP 精准定位

## 工作流程

1. **搜索像素参考** — `"<对象> pixel art sprite 32x32"` 优先，实物照片仅补充
2. **分层设计** — 轮廓 → 结构 → 细节 → 色彩，先展示设计再动手
3. **自动选尺寸** — 简单(16×16) / 中等(24×24) / 复杂(32×32) / 精细(48×48)
4. **逐格绘制** — CDP 浏览器 + 颜色分组批量点击
5. **复盘** — 轮廓识别度？特征抓住了吗？能否改进？

## 设计原则

- **少即是多** — 32×32 仍然很小，优先保证轮廓识别度
- **颜色经济** — 3-6 色最佳，每种颜色有明确语义
- **符号化** — 像素画不是照片，抓住 3 个关键特征即可

## 技术栈

- 纯 HTML/CSS/JS（画板页面）
- CDP (Chrome DevTools Protocol) 鼠标操控
- WebSearch 参考图检索
- Node.js 24+（CDP Proxy）
